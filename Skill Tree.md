*Shaped like a four leafed clover :)*
you spend [[Luck]] in the Skill Tree

There are 5 Branches in the Skill Tree:

## Blade Master (Swords)
This path Boosts Swords and has a focus on [[Weapons#Attributes|Attributes]]
## Brawler (Gauntlets)
This path Boosts Gauntlets and has a focus on **Crit chance** and **Crit multiplier** 
## Bulwark (Shield)
This path boosts Shields and has a focus on **Health** and **Defense**
## Berserker (Axes)
This path boosts Axes and has a focus on **Big Damage**
## Basic (General Stats)
This path just gives general stat upgrades
